Definition of Vector Space

Vector Space​

  • EEE to be a non-null set and KKK to be a scalar set
  • Vectors: elements of the set EEE
  • +++ - Internal composition law, E×E→EE\times E \to EE×E→E
  • ... - External composition law, K×E→EK\times E\to EK×E→E
  • (E,+,.E,+,.E,+,.) is said vector space of the vector set EEE over the scalar field (K,+,K,+,K,+,) if and olf if the ten vector space axioms are verified

Axioms​

  • EEE is closed with respect to the internal composition law: ∀u,v∈E:u+v∈E∀u, v ∈ E : u + v ∈ E∀u,v∈E:u+v∈E
  • EEE is closed with respect to the external composition law: ∀u∈Eand∀λ∈K:λu∈E∀u ∈ E and ∀λ ∈ K : λu ∈ E∀u∈Eand∀λ∈K:λu∈E
  • Commutativity for the internal composition law: ∀u,v∈E:u+v=v+u∀u, v ∈ E : u + v = v + u∀u,v∈E:u+v=v+u
  • Associativity for the internal composition law: ∀u,v,w∈E×E:u+(v+w)=(u+v)+w∀u, v, w ∈ E × E : u + (v + w) = (u + v) + w∀u,v,w∈E×E:u+(v+w)=(u+v)+w
  • Neutral element for the internal composition law: ∀u∈E:∃!o∈E∣u+o=u∀u ∈ E : ∃!o ∈ E|u + o = u∀u∈E:∃!o∈E∣u+o=u
  • Opposite element for the internal composition law: ∀u∈E:∃!−u∈E∣u+−u=o∀u ∈ E : ∃!−u ∈ E|u + −u = o∀u∈E:∃!−u∈E∣u+−u=o
  • Associativity for the external composition law: ∀u∈E∀u ∈ E∀u∈E and ∀λ,μ∈K:λ(μu)=(λμ)u=λμu∀λ, μ ∈ K : λ(μu) = (λμ) u = λμu∀λ,μ∈K:λ(μu)=(λμ)u=λμu
  • Distributivity 1: ∀u,v∈E∀u, v ∈ E∀u,v∈E and ∀λ∈K:λ(u+v)=λu+λv∀λ ∈ K : λ (u + v) = λu + λv∀λ∈K:λ(u+v)=λu+λv
  • Distributivity 2: ∀u∈Eand∀λ,μ∈K:(λ+μ)u=λu+μu∀u ∈ E and ∀λ, μ ∈ K : (λ + μ)u = λu + μu∀u∈Eand∀λ,μ∈K:(λ+μ)u=λu+μu
  • Neutral elements for the external composition law: ∀u∈E:∃!1∈K∣1u=u∀u ∈ E : ∃!1 ∈ K |1u = u∀u∈E:∃!1∈K∣1u=u

Vector Subspace​

(E,+,.E,+,.E,+,.) be a vector space, U⊂EU \subset EU⊂E and U≠∅U \ne ∅U=∅ The triple (E,+,.E,+,.E,+,.) is a vector subspace of (E,+,.E,+,.E,+,.) if (U,+,.U,+,.U,+,.) is a vector space over the same field KKK with respect to both the composition laws

Proposition shows that we do not need to prove all 10 axioms, just need to prove closure of the two composition laws.

Null vector in Vector Spaces​

(E,+,.E,+,.E,+,.) be a vector space over a field KKK. Every vector subspace (U,+,.U,+,.U,+,.) of (E,+,.E,+,.E,+,.) contains the null vector. For every vector space (E,+,.E,+,.E,+,.), at least two vector subspace exist

(U,+,.U,+,.U,+,.) and (V,+,.V,+,.V,+,.) be a vector space of (E,+,.E,+,.E,+,.). Sum subset is a set S=U+VS=U+VS=U+V defined as S=U+V={w∈E∣∃u∈U,v∈V∣w=u+v}S=U+V=\{ w\in E |\exists u \in U, v \in V|w=u+v \}S=U+V={w∈E∣∃u∈U,v∈V∣w=u+v}
