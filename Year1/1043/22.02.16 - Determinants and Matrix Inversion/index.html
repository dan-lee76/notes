Properties of determinants

Determinant of a matrix AAA is equal to the determinant of its transpose matrix

  • Transpose Matrix: detA=detATdetA = det A^TdetA=detAT
  • Triangular Matrix: Equal to the product of the diagonal elements
  • Row(Column) swap: If two rows (columns) are swapped the determinant of the matrix AsA_sAs​ is −detA-detA−detA
  • Null determinant: if two rows(columns) are identical(sum) then detA=0detA=0detA=0
    • If and only if the rows(columns) are linearly dependent
    • if and only if at least one row(column) is a linear combination of the other rows(columns)
    • if a row(column) is proportional to another row(column)
  • Invariant Determinant: Row(column) the elements of another row(column) all multiplied by the same scalar are added, the determinant remains the same
  • Row(column) multiplied by a scalar: Row(column) is multiplied by λ\lambdaλ then λdetA\lambda detAλdetA
  • Matrix multiplied by a scalar: If λ\lambdaλ is scalar, det(λA)=λndetAdet(\lambda A) = \lambda ^n detAdet(λA)=λndetA
  • Determinant of the product: Product between two matrices is equal to the products of the determinants. det(AB)=det(A)det(B)=det(B)det(A)=det(BA)det(AB) = det(A)det(B) = det(B)det(A) = det(BA)det(AB)=det(A)det(B)=det(B)det(A)=det(BA)
